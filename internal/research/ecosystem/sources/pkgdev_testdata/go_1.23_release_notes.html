Go 1.23 Release Notes

Changes to the language

Go 1.23 makes the (Go 1.22) "range-over-func" experiment a standard part of the language. The "range" clause in a "for-range" loop now accepts iterator functions of the following types as range expressions.

Tools

Go telemetry

Starting in Go 1.23, the Go toolchain can collect usage and breakage statistics that help the Go team understand how the toolchain is used and how well it is working.

Go command

Setting the GOROOT_FINAL environment variable no longer has an effect. Distributions that install the go command to a location other than $GOROOT/bin/go should install a symlink instead of relocating or copying the go binary.

Runtime

The runtime now contains experimental support for cooperative goroutine cancellation and improved scheduling of concurrent operations. The traceback printer has been improved to format inlined frames more clearly.

Compiler

The build time overhead to building with Profile Guided Optimization has been reduced significantly.

Standard library

New iter package

The new iter package provides the basic definitions for working with user-defined iterators.

New structs package

The new structs package provides types for struct fields that modify properties of the containing struct type such as memory layout.

New unique package

The new unique package provides facilities for canonicalizing values (like "interning" or "hash-consing").

Minor changes to the library

The following packages have been deprecated: archive/tar's experimental Format. Several APIs have been removed in favor of generic alternatives.
