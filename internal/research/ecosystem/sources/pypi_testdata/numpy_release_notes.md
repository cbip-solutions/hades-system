# NumPy 1.26.0 Release Notes

NumPy 1.26.0 adds new features and bug fixes.

## New Features

- Added `numpy.linalg.matrix_power` overload for sparse matrices.
- Added `numpy.fft.fft2` parallel backend option.

## Deprecations

- Deprecated `numpy.distutils`; will be removed in 2.0.

## Bug Fixes

- Fixed `numpy.argmax` edge case for empty arrays.
